◆ NewBag()

void CWorkshopController::NewBag ( )

Select a new bag

Call this function to select a new bag. The new bag and name are selected and the bag is set on the end of the conveyer, which is off the screen. You should then set the speed to -1 to get the bag to roll into the loading area.
