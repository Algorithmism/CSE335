Detailed Description

Workshop simulation class

This class simulates a workshop at a secret location that manages delivery of toys for a certain unnamed individual.

You do not directly access the functions in CWorkshopWnd. Instead, use the functions in CWorkshopController for managing all Workshop operations.

Update the simulation based on time

This will advance in millisecond increments only.
